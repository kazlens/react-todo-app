# React To-Do App

Мини-проект: список задач на React.

## Что есть в проекте

- добавление, редактирование и удаление задач;
- отметка выполненных задач;
- поиск и фильтр;
- сохранение задач в `localStorage`;
- загрузка задач с Placeholder API;
- две страницы через React Router;
- светлая и темная тема.

## Страницы

- `/` - список задач;
- `/api` - задачи, полученные с JSONPlaceholder.

## Запуск

```bash
npm install
npm run dev
```
